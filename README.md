Projeto desenvolvido em JavaScript com foco na criação de um sistema interativo que sorteia hobbies de forma aleatória, apresentando ao usuário novas atividades físicas e criativas para experimentar.

A aplicação exibe uma imagem, uma descrição estética e leve, e um background dinâmico para cada hobby selecionado — tudo acompanhado de animações suaves que tornam a experiência mais envolvente.
A ideia é incentivar as pessoas a explorarem novas formas de movimento, descobrirem atividades que combinem com seu estilo de vida e quebrarem a rotina com algo diferente.

O projeto foi desenvolvido como prática de manipulação do DOM, animações em CSS, eventos em JavaScript, além de trabalhar conceitos de responsividade e experiência do usuário.
